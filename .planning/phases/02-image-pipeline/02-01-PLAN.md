Create `flyer_generator/stages/preprocessor.py` with:

1. **ImagePreprocessor class** (per D-13, D-14, D-15):
   ```python
   from PIL import Image
   import io

   class ImagePreprocessor:
       SOURCE_DIMENSIONS = (832, 1472)
       FINAL_DIMENSIONS = (1080, 1920)

       def upscale(self, raw_bytes: bytes, comfy_job: ComfyJob) -> GeneratedBackground:
           """Upscale raw 832x1472 PNG bytes to 1080x1920."""
           img = Image.open(io.BytesIO(raw_bytes))
           source_dimensions = img.size  # (width, height)
           resized = img.resize(self.FINAL_DIMENSIONS, Image.Resampling.LANCZOS)
           buf = io.BytesIO()
           resized.save(buf, format="PNG")
           return GeneratedBackground(
               image_bytes=buf.getvalue(),
               source_dimensions=source_dimensions,
               final_dimensions=self.FINAL_DIMENSIONS,
               comfy_job=comfy_job,
           )
   ```

   Note: The method takes `comfy_job: ComfyJob` as a parameter to populate the GeneratedBackground model (per D-13 it returns GeneratedBackground). The ComfyJob is passed through from the ComfyClient which creates it during submit.
