2. **VisionEvaluator class** (per D-16 through D-22):

```python
import base64
import json
import re
import structlog
from anthropic import AsyncAnthropic, APIError
from pydantic import ValidationError

logger = structlog.get_logger()
```

- **`__init__(self, settings: Settings)`** (per D-16 -- using Anthropic SDK as recommended discretion choice):
  - `self._settings = settings`
  - `self._client = AsyncAnthropic(api_key=settings.anthropic_api_key.get_secret_value(), timeout=settings.vision_timeout_seconds)`

- **`async def evaluate(self, background: GeneratedBackground, event: EventInput) -> VisionVerdict`** (per D-16, D-18):
  - Build user text: `f"Event: {event.title}\nDate: {event.date}\nTime: {event.time}\nVenue: {event.location_name}\nAddress: {event.location_address}\nFees: {event.fees}\nOrganizer: {event.org}\nStyle: {event.style_concept}"`
  - Base64 encode: `img_b64 = base64.b64encode(background.image_bytes).decode()` (per D-18, encode at point of use, don't store)
  - Call SDK (per research Pattern 2):
    ```python
    response = await self._client.messages.create(
        model=self._settings.vision_model,
        max_tokens=self._settings.vision_max_tokens,
        system=VISION_SYSTEM_PROMPT,
        messages=[{
            "role": "user",
            "content": [
                {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": img_b64}},
                {"type": "text", "text": user_text},
            ],
        }],
    )
    ```
  - Extract raw text: `raw_text = response.content[0].text`
  - Try parsing: `return self._parse_and_validate(raw_text)`
  - On `VisionResponseParseError`: one retry (per D-20)
    ```python
    retry_response = await self._client.messages.create(
        model=self._settings.vision_model,
        max_tokens=self._settings.vision_max_tokens,
        system=VISION_SYSTEM_PROMPT,
        messages=[
            {"role": "user", "content": [
                {"type": "image", "source": {"type": "base64", "media_type": "image/png", "data": img_b64}},
                {"type": "text", "text": user_text},
            ]},
            {"role": "assistant", "content": raw_text},
            {"role": "user", "content": "Return valid JSON only, no prose."},
        ],
    )
    raw_text_retry = retry_response.content[0].text
    return self._parse_and_validate(raw_text_retry)
    ```
  - On `APIError` from SDK: raise `VisionAPIError(str(e))` wrapping the SDK error
  - Log: `logger.info("vision_evaluated", approved=verdict.approved, confidence=verdict.confidence)`

- **`def _parse_and_validate(self, raw_text: str) -> VisionVerdict`** (per D-19, D-21, D-22):
  - Step 1: Strip markdown fences: `cleaned = re.sub(r"```json\s*", "", raw_text, flags=re.IGNORECASE)` then `cleaned = re.sub(r"```\s*$", "", cleaned).strip()`
  - Step 2: Extract first `{` to last `}`: find indices, raise `VisionResponseParseError` if not found
  - Step 3: `json.loads(json_str)` -- on `json.JSONDecodeError`, raise `VisionResponseParseError`
  - Step 4 -- Confidence gate (per D-21): if `data.get("approved")` and `confidence < self._settings.vision_confidence_threshold`, flip `approved` to `False` and append `f"Low confidence: {data['confidence']}"` to rejection_reasons
  - Step 5 -- Zone validation (per D-22): if `data.get("approved")` and zones is null or any zone name not in ZoneName values, flip `approved` to `False` and append reason. Use `get_args(ZoneName)` from `typing` to get valid zone names.
  - Step 6: `VisionVerdict(**data, raw_response=raw_text[:500])` -- on `ValidationError`, raise `VisionResponseParseError`

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client -> Anthropic API | API key in SDK, response text is untrusted LLM output |
| LLM response -> VisionVerdict | Raw text parsed into structured data -- primary injection surface |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-08 | S (Spoofing) | Anthropic API key | mitigate | API key stored as SecretStr in Settings, passed to SDK which manages auth header |
| T-02-09 | T (Tampering) | Vision response JSON | mitigate | Parse through json.loads + Pydantic validation -- no eval() or exec(). Zone names validated against ZoneName Literal. raw_response truncated to 500 chars. |
| T-02-10 | I (Info Disclosure) | base64 image in request | accept | Image sent to Anthropic API is generated art (not user PII). Standard API TOS applies. |
| T-02-11 | D (Denial of Service) | parse retry loop | mitigate | Single retry only (not unbounded). Timeout on SDK calls via settings.vision_timeout_seconds. |
| T-02-12 | E (Elevation) | raw_response field | mitigate | raw_response stored truncated to 500 chars to prevent memory amplification from adversarial responses |
</threat_model>
