Create `flyer_generator/pipeline.py` with the FlyerGenerator class per D-01 through D-06, D-16, D-17:

1. **Constructor** (per D-01, D-06):
   ```python
   class FlyerGenerator:
       def __init__(
           self,
           settings: Settings,
           presets: PresetRegistry | None = None,
           http_client: httpx.AsyncClient | None = None,
       ) -> None:
   ```
   - If `presets` is None, call `build_default_registry()`
   - If `http_client` is None, create `httpx.AsyncClient()` and store `self._owns_http = True`
   - Create all stage instances in __init__:
     - `self._prompt_builder = StylePromptBuilder(presets)`
     - `self._comfy_client = ComfyClient(settings, http_client)`
     - `self._preprocessor = ImagePreprocessor()`
     - `self._vision = VisionEvaluator(settings)`
     - `self._layout = LayoutResolver()`
     - `self._composer = PosterComposer()`
     - `self._rasterizer = Rasterizer()`
   - Store `self.settings = settings`

2. **generate() method** (per D-02, D-03, D-04, D-05):
   ```python
   async def generate(self, event: EventInput) -> FlyerOutput:
   ```
   - Generate `trace_id = uuid.uuid4().hex` (per D-04)
   - Get logger via `get_logger().bind(trace_id=trace_id, event_title=event.title)`
   - Initialize `refinement_hint = ""` and `rejection_history: list[str] = []`
   - Loop `for attempt in range(1, self.settings.max_bg_attempts + 1):`
     - Log `attempt_start` with attempt number and hint (per D-16)
     - `workflow = self._prompt_builder.build(event, attempt, refinement_hint)`
     - Log prompt hash at info, full prompt at debug only (per D-17): `logger.info("comfy_submitted", prompt_hash=hashlib.sha256(workflow.positive_prompt.encode()).hexdigest()[:12])` -- but note comfy_client already logs comfy_submitted with prompt_id, so log prompt_hash alongside
     - `comfy_job, raw_bytes = await self._comfy_client.generate(workflow, attempt)` -- CRITICAL: generate() returns tuple[ComfyJob, bytes], NOT GeneratedBackground
     - `background = self._preprocessor.upscale(raw_bytes, comfy_job)`
     - `verdict = await self._vision.evaluate(background, event)`
     - If `verdict.approved`:
       - Log `vision_approved` with confidence and zones.model_dump() (per D-16)
       - `layout = self._layout.resolve(verdict.zones)`
       - `svg = self._composer.compose(event, background, verdict, layout)`
       - `png_bytes = self._rasterizer.rasterize(svg)`
       - Log `flyer_generated` with `len(png_bytes) // 1024` and attempt count (per D-16)
       - Return `FlyerOutput(png_bytes=png_bytes, dimensions=(1080, 1920), file_size_kb=len(png_bytes) // 1024, event_title=event.title, attempts_used=attempt, final_vision_verdict=verdict, zones_used=verdict.zones, trace_id=trace_id)`
     - Else (rejected):
       - Log `vision_rejected` with attempt, reasons, and hint (per D-16)
       - `rejection_history.extend(verdict.rejection_reasons)`
       - `refinement_hint = verdict.refinement_hint`
   - After loop: `raise MaxAttemptsExceededError(f"Vision rejected {self.settings.max_bg_attempts} backgrounds. Rejection history: {rejection_history}", trace_id=trace_id)`

3. **Imports:** uuid, hashlib, httpx, structlog, all stage classes, all models, Settings, PresetRegistry, build_default_registry, MaxAttemptsExceededError, get_logger.

4. **Do NOT** log full prompts at info level, base64 payloads, or API keys (per D-17).
