   ```python
       org: Annotated[str | None, typer.Option(help="Organizer name")] = None,
       concept: Annotated[str | None, typer.Option(help="Style concept for image generation")] = None,

       event_json: Annotated[Path | None, typer.Option("--event-json", help="Load event from JSON file")] = None,
       list_presets: Annotated[bool, typer.Option("--list-presets", help="List available presets")] = False,
       dry_run: Annotated[bool, typer.Option("--dry-run", help="Print prompt without generating")] = False,
       max_attempts: Annotated[int | None, typer.Option("--max-attempts", help="Max background generation attempts")] = None,
   ) -> None:
   ```

   - `configure_logging(settings.log_format, settings.log_level)`
   - `generator = FlyerGenerator(settings)`
   - `result = asyncio.run(generator.generate(event))`
   - `result.save(output)`
   - Print success message with file path and attempts used.

8. **Error handling:** Wrap the generate call in try/except FlyerGeneratorError, print user-friendly error message, `raise typer.Exit(code=1)`.
