Title sizing (D-06 -- measure AFTER .upper()):
  len <= 14: fontSize=82, maxCharsPerLine=14
  len <= 20: fontSize=72, maxCharsPerLine=20
  len <= 30: fontSize=62, maxCharsPerLine=18
  else:      fontSize=52, maxCharsPerLine=22

Title line height: fontSize * 1.25
Title block vertical center: titleStartY = layout.title.y - (titleTotalHeight / 2) + fontSize * 0.8
Title font: 'Arial Black', 'Helvetica Neue', Arial, sans-serif; weight=900; letter-spacing=2
Title stroke: paint-order="stroke" stroke=strokeColor stroke-width="3"

Text colors (D-08):
  white: textColor=#ffffff, strokeColor=rgba(0,0,0,0.5), subtleTextColor=rgba(255,255,255,0.85)
  dark:  textColor=#1a1a1a, strokeColor=rgba(255,255,255,0.4), subtleTextColor=rgba(26,26,26,0.75)

Scrim fade color (D-09):
  white text: fadeColor='0,0,0' (darken)
  dark text:  fadeColor='255,255,255' (lighten)

Scrim zones: extract row from zone name via split('_')[0]. Only add scrims for rows used by title and details.
  TOP: rect x=0 y=0 w=1080 h=700 fill=url(#topFade)
  BOTTOM: rect x=0 y=1220 w=1080 h=700 fill=url(#bottomFade)
  MIDDLE: rect x=0 y=600 w=1080 h=720 fill=url(#middleFade)

Accent line (D-12): 200x4px rect, rx=2, fill=accentColor, opacity=0.9
  Positioned at titleBlockBottom = titleStartY + (titleLines.length - 1) * lineHeight + 20
  X based on anchor: middle -> title.x - 100, start -> title.x, end -> title.x - 200

Details block (from n8n, offsets from layout.details.y):
  date:      y = dy - 40, font='Arial Black', size=40, bold, letter-spacing=1
  time:      y = dy + 15, font=Arial, size=34, fill=subtleTextColor
  separator: y = dy + 42, rect w=140 h=2 rx=1, fill=textColor opacity=0.3
  venue:     y = dy + 90, font=Arial, size=30, bold, opacity=0.95
  address:   y = dy + 130, font=Arial, size=24, fill=subtleTextColor
  url:       y = dy + 210, font=Arial, size=22, fill=subtleTextColor (only if url present)

Separator X based on anchor:
  middle: dx - 70
  end:    dx - 140
  start:  dx

Fee badge (D-10):
  width = clamp(len(fees) * 22 + 60, 140, 400)
  height = 56, rx = 28 (full pill)
  font size: 22 if len(fees) > 15 else 30
  text fill: always #1a1a1a
  badge X based on anchor: middle -> fx - width/2, start -> fx, end -> fx - width
  badge Y: fy - 28 (centered on zone y)
  text Y: fy + 8

Org credit (D-11): x=540, y=1840, text-anchor=middle, font=Arial size=20, fill=textColor opacity=0.55
  Text: "Presented by {org}"

Accent stripe (D-12): rect x=0 y=1908 w=1080 h=12 fill=accentColor opacity=0.8

SVG wrapper: xmlns + xmlns:xlink, width=1080 height=1920 viewBox="0 0 1080 1920"
Background: <image href="data:image/png;base64,{...}" width="1080" height="1920" preserveAspectRatio="xMidYMid slice"/>

Gradient defs:
  topFade: linearGradient y1=0 y2=1, stop 0%=fadeColor@0.75, stop 100%=fadeColor@0
  bottomFade: linearGradient y1=0 y2=1, stop 0%=fadeColor@0, stop 100%=fadeColor@0.85
  middleFade: radialGradient cx=0.5 cy=0.5 r=0.6, stop 0%=fadeColor@0.6, stop 100%=fadeColor@0
